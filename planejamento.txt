Desafio
- Criar Usuário (Terá duas Permissões de Usuários > Comum e o Master) - OK

- Cada Usuário poderá incluir cadastrar pacientes - OK

- Cada Usuário poderá atualizar os dados do paciente - OK

- Cada Usuário poderá incluir Evolução de Paciente - OK

- Cada Paciente terá 3 estágios: Cancelado, Renunciado, Ativo - OK

- Criar Filtro de Status dos Pacientes e Valor Total de Pacientes Cadastrados - PROXIMO

- Criar Filtro Data de Evolução de Paciente

- Criar Validação na Rota de DELETE, para apenas o "User Master" deletar pacientes - OK
